# Web Server Using Docker

## Project Description
This project demonstrates how to create and deploy a simple web server using Docker and Nginx.

## Technologies Used
- Docker
- Nginx
- HTML

## Project Files
- Dockerfile
- index.html

## Steps Performed
1. Created a simple HTML web page.
2. Created a Dockerfile using Nginx image.
3. Built a Docker image using Docker commands.
4. Created and ran a Docker container.
5. Verified the web server in the browser using localhost.
